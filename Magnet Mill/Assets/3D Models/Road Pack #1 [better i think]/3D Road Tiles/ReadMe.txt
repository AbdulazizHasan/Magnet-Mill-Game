To hide the annoying .mtl files, run the following command using Bash (linux shell) in this directory (you need to "cd" to this dir) [3D Road Tiles]
	example directory: "Z:\Unity Projects\Magnet-Mill-Game\Magnet Mill\Assets\3D Models\Road Pack #1 [better i think]\3D Road Tiles" - [Windows]
			   "/mnt/z/Unity Projects/Magnet-Mill-Game/Magnet Mill/Assets/3D Models/Road Pack #1 [better i think]/3D Road Tiles" - [Linux]

Command to run Bash Script to execute in order to hide .MTL files:
./hideMTL.sh
